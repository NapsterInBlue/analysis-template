Features
--------

- Workflows optimized for CI/CD and Version Control best practices
- Robust project structure with Python pathing resolved, out of the box
- Extensive Makefile automation including

    - Virtual Environment setup and teardown
    - Collection and processing of source data
    - Timestamped and watermarked report generation
